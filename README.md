# ClassPulse360

ClassPulse 360 is a real-time classroom engagement and assessment platform that enables teachers to create quizzes, conduct live interactive sessions, run classroom assessments, and measure student understanding instantly through comprehensive analytics and leaderboards.

## Project Structure

- `frontend/` - Frontend application
- `backend/` - FastAPI backend application
- `supabase/` - Database migrations, configuration, and seed scripts
- `docs/` - Project documentation
- `docker/` - Docker deployment configurations
- `.github/` - CI/CD workflows
